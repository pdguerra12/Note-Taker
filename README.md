# Note-Taker

## Purpose

To provide the user an application to write and save notes. It uses Express.js back end and will save and retrive note data from a JSON file.

---

## Technology Used

- HTML
- CSS
- Javascript
- Node.js
- Inquirer
- FS
- JSON
- Heroku
